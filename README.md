# Cake Quests

  Lightweight Minecraft mod that adds a progression quest book similar to FTB Quests.
  Designed to have as few dependencies as possible and support two modes:

  - the server also has the mod, which enables quest progress tracking
  - the server does not have the mod, which disables progression tracking but still lets users use the book as a guide through complicated modpacks

  Actual quests are not shipped with this mod, as they are highly dependent on the mods installed on a server.
  There will be an auxiliary mod that provides the quest book for Vault Hunters Third Edition, which is the main reason for creating this mod.

## Credits

Credits for most of the hard work go to the FTB Quests team. This is just a slightly different, more lightweight version that removes the dependencies on FTB and targets a slightly different use case.
